# Home Page (src/app/page.tsx)

## Responsabilidades:
- Implementar a Hero Section com imagem aspiracional e copy impactante.
- Exibir um grid de produtos em destaque consumindo a API (/produtos).
- Seção "Sobre a Marca" detalhando a proposta de valor premium e natural.

## Estilo:
- Uso generoso de whitespace (espaçamento) para luxo e calma.
- Grid de produtos com Flexbox (flex-wrap e gap).
- Micro-interações sutis nos cards de produto (hover transitions lentas).
