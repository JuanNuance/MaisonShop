# App Layout (src/app/layout.tsx)

## Responsabilidades:
- Configurar o RootLayout para a aplicação Next.js.
- Importar e configurar as fontes Serifada (Playfair Display/Lora) e Sans-serif (Inter/Satoshi).
- Implementar o background Stone-50 ou Off-white globalmente.
- Configurar o Context Provider para Gerenciamento de Estado (React Context ou Zustand).
- Configurar o QueryClientProvider para o TanStack Query.
- Incluir os componentes Navbar e Footer em todas as páginas.

## Estilo:
- Utilizar Tailwind CSS com foco em variáveis para cores (Stone-900, Sage/Olive).
- Garantir que o layout seja responsivo usando puramente Flexbox.
