# Common Components (src/components/common/common.md)

## Componentes de Layout:
- **Navbar:** Menu hambúrguer sofisticado, logo centralizado (em fonte serifada) e links de navegação.
- **Footer:** Rodapé completo com links institucionais e newsletter.
- **Logo:** Componente centralizado para a marca com tipografia Playfair Display.
- **Container:** Wrapper para limitar a largura máxima do conteúdo de forma responsiva.
