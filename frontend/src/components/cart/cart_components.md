# Cart Components (src/components/cart/cart_components.md)

## Componentes de Carrinho:
- **CartDrawer (Sheet):** Painel lateral deslizante para visualizar o carrinho em qualquer página.
- **CartItem:** Linha individual do item no carrinho com imagem, nome, preço e controle de quantidade.
- **CartSummary:** Resumo financeiro (Subtotal, Taxas, Total).
