# UI Components (src/components/ui/ui.md)

## Componentes Atômicos Reutilizáveis:
- **Button:** Diferentes estados (primary, secondary, outline) com transições lentas.
- **Input:** Campos de texto minimalistas com foco em tipografia Inter/Satoshi.
- **Badge:** Indicadores de categoria ou status (disponível/esgotado).
- **Card:** Base para produtos e informações com bordas levemente arredondadas e sombras suaves.
- **Skeleton:** Loaders baseados em pulsos suaves para listagens e detalhes.

## Estilo:
- Bordas: `rounded-sm` ou `rounded-md`.
- Sombras: `shadow-sm`.
- Transições: `transition-all duration-300 ease-in-out`.
