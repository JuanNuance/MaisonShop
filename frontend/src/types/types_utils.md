# Types & Utils (src/types/types.md e src/utils/utils.md)

## Tipagem (Types):
- Interfaces TypeScript rigorosas para `Produto`, `Usuario`, `Carrinho`, `Pedido`.
- Definição de tipos para respostas de API e erros.

## Utilitários (Utils):
- **formatPrice:** Função para formatar valores monetários em BRL.
- **dateUtils:** Formatação de datas para histórico de pedidos.
- **storage:** Wrappers para LocalStorage/Cookies.
