# State Management (src/store/store.md)

## Responsabilidades:
- Configuração do Zustand ou React Context para gerenciamento de estado global.
- Persistência básica para o carrinho de compras (Local Storage ou Cookie).
- Estado global para:
    - Carrinho (CartState)
    - Usuário autenticado (AuthState)
    - UI (Modal/Drawer state)
